친구

1058: 친구

문제

지민이는 세계에서 가장 유명한 사람이 누구인지 궁금해졌다. 가장 유명한 사람을 구하는 방법은 각 사람의 2-친구를 구하면 된다. 어떤 사람 A가 또다른 사람 B의 2-친구가 되기 위해선, 두 사람이 친구이거나, A와 친구이고, B와 친구인 C가 존재해야 된다. 여기서 가장 유명한 사람은 2-친구의 수가 가장 많은 사람이다. 가장 유명한 사람의 2-친구의 수를 출력하는 프로그램을 작성하시오.

A와 B가 친구면, B와 A도 친구이고, A와 A는 친구가 아니다.

입력

첫째 줄에 사람의 수 N이 주어진다. N은 50보다 작거나 같은 자연수이다. 둘째 줄부터 N개의 줄에 각 사람이 친구이면 Y, 아니면 N이 주어진다. (예제를 참고)

출력

첫째 줄에 가장 유명한 사람의 2-친구의 수를 출력한다.

예제 입력

3
NYY
YNY
YYN

예제 출력

2

해설

문제의 이해를 위해 다음과 같은 샘플 데이터를 받았다고 가정 해 보자,

5
NYYNN
YNYNN
YYNYN
NNYNY
NNNYN

이러한 데이터를 그래프로 표현하면 다음과 같다.

위 그래프에서 1번 노드(사람)와 '2-친구' 관계인 노드는 2, 3, 4번 노드이다.

[2, 3번 노드] = 1번과 직접 아는 친구

[4번 노드] = 3번 노드를 통해 함께 아는 친구

이러한 사실을 통해 문제를 이해한 후, 문제 해결을 위해 작성한 코드는 다음과 같다.

string person[51];
int two_friends(int p) {
     int ret = 0;
     bool visit[50] = { 0, };
     visit[p] = true;
     for (int i = 0; i < n; i++) {
          if (person[p][i] == 'Y') {
               if (visit[i] == false) ret++;
               visit[i] = true;
               for (int j = 0; j < n; j++) {
                    if (person[i][j] == 'Y' && visit[j] == false) {
                         visit[j] = true;
                         ret++;
                    }
               }
          }
     }
     return ret;
}

위 two_friends 함수는 특정 사람의 2-친구 수를 return하는 함수이다.

ret 변수는 최종적으로 p와 친구인 사람의 수를 저장하기 위한 변수이다.

visit 변수는 친구의 수를 셀 때 중복되는 경우를 방지하기 위해 선언 한 변수이며, 자기 자신과는 친구가 될 수 없기 때문에 미리 자기 자신을 true로 지정한다.

p 번 사람과 친구가 될 수 있는 사람은 (자기 자신을 포함하여) n명 이므로, n번에 걸쳐 루프를 돌게 된다.

이 루프에서는 p의 2-친구의 수를 구해서 ret변수에 저장하게 된다.

우선 p 와 i 가 친구일 경우, i가 이미 친구로 선택된 적이 있는지를 확인하게 된다. 지금까지 친구로 선택된 적이 없었다면, ret에 1을 더하고 하고 visit변수를 최신화하여 친구로 선택했음을 표시한다.

그 이후 2-친구의 2번째 조건인 친구의 친구의 존재를 확인하기 위해 한차례 더 루프를 돌게 된다.

만약 i가 j와 친구이고, j가 아직 친구로 선택이 된 적이 없다면 visit를 최신화 하여 j를 친구로 선택했음을 표시하고 ret에 1을 더하게된다.

이러한 과정을 통해서 two_friends 함수에서는 파라미터로 주어진 인덱스의 사람의 2-친구의 수를 구하게 된다.

main()
{
     cin >> n;
     for (int i = 0; i < n; i++) cin >> person[i];
     int max_ = -1;
     for (int i = 0; i < n; i++) {
          int k = two_friends(i);
          if (max_ < k) max_ = k;
     }
     cout << max_ << endl;
}

위의 two_friends 함수를 활영하여, main 함수에서 문제의 답을 구한다.

max_ 변수와 for문을 통해 Greedy 방식으로 가장 큰 2-친구의 수를 구하고, 이를 출력한다.
